Introduction

Welcome to MyProject — a fast, flexible framework for building modern web applications. This guide walks you through everything you need to get started.

ℹ️

New in v2.4 — We've revamped the caching system and added first-class support for streaming. See what's new →

Quick Start

The fastest way to create a new MyProject app is using the CLI. Run the following command and follow the prompts:

        

          Terminal
          
        

        npx create-myproject@latest my-app
cd my-app
npm run dev

Your app will be available at http://localhost:3000. Open the project in your editor and start building.

💡

Tip: Use the --template flag to start from a pre-built template. Run npx create-myproject --help to see all options.

Manual Installation

Prefer to set things up yourself? Follow these steps to install MyProject manually in an existing project.

1

Install dependencies

Add MyProject and its peer dependencies to your project.

              

                Terminal
                
              

              npm install myproject react react-dom
2

Create a config file

Add a myproject.config.js file at the root of your project.

              

                myproject.config.js
                
              

              /** @type {import('myproject').Config} */
const config = {
  port: 3000,
  output: 'standalone',
  features: {
    streaming: true,
    caching: 'auto',
  },
};

export default config;
3

Add scripts to package.json

Update your package.json to include the dev and build scripts.

              

                package.json
                
              

              {
  "scripts": {
    "dev": "myproject dev",
    "build": "myproject build",
    "start": "myproject start"
  }
}

System Requirements

Before installing MyProject, make sure your environment meets the minimum requirements.

Dependency Minimum Version Recommended
Node.js 18.17.0 20.x or later
npm 9.x 10.x or later
OS macOS, Linux, Windows (WSL) —
⚠️

Windows users: We recommend using Windows Subsystem for Linux (WSL) for the best development experience.
